TODAY-FIRST TODO PLATFORM

目標を、今日やるTodoに落とし込む。

Letodo は、人生や仕事の目標を日々のTodoに変換し、 Today画面で迷わず実行できる状態までつなげる Todo SaaS です。

GitHub を見る 開発状況を見る
Today First

その日にやるべきことを、1画面で明確にする。

Goal to Action

抽象的な目標を、実行できる粒度のTodoまで分解する。

Reflection Loop

完了だけでなく、未完了の整理と振り返りまで扱う。

Problem

目標はあるのに、今日やることに落ちない。

目標が抽象的なまま止まる

長期の目標は立てられても、今日どの行動を選べばいいかが見えない。

タスクが複数ツールに散らばる

カレンダー、メモ、タスク管理ツールに情報が分散し、全体像を見失いやすい。

振り返りまで続かない

1日の終わりに整理する仕組みが弱く、未完了が積み上がっていく。

Value

Letodo が支える 3 つの流れ

01

目標を行動に変える

Mission、Goal、Todo の階層で考えを整理し、抽象的な意図を今日の実行へ変える。

02

Today で実行に集中する

予定、習慣、持ち越し、外部連携タスクを Today に集約し、その日にやることを決めやすくする。

03

振り返りまで 1 日として扱う

完了だけでなく、延期、中止、持ち越し、メモまで含めて 1 日の体験を閉じる。

Why Now

Todo 管理ではなく、実行の文脈をつなぐ。

Letodo は「タスクを並べるだけ」の道具ではなく、目標分解、Today、 振り返り、外部ツールの集約を 1 つの流れで扱うことを目指しています。

Status

現在はベータ版 MVP の土台づくりを進行中です。

まずは「目標と今日の行動がつながること」と、 「Today で迷わず実行できること」を中心に絞って開発しています。

現在の主軸 Today 中心の実行体験
